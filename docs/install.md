# Installation

## Install

---
You can use

```bash
curl -fsSL https://masterrc.master3307.org/ | bash
```

to *install* it with one command.

!!! warning "Review before running"
    Please review this code before you install it. Don't blame me for any damage!
    It should be 100% safe, but [you never know](https://cheatsheetseries.owasp.org/cheatsheets/Secure_Code_Review_Cheat_Sheet.html).
    View the code [here](https://masterrc.master3307.org/).

## Update

---
You can update it by running `masterrc` in a Console Window or the install command again.

!!! info "Info"
    Running `aptt`, `updates` or `upgrades` also updates the MasterRC script among other things. Check the **[Usage](usage.md)** page for more information on the commands.

## Uninstall

---

Run `masterrc-uninstall` in a Terminal Window to uninstall.

optionally you can use

```bash
curl -fsSL https://u-masterrc.master3307.org/ | bash
```

to *uninstall* it with one command.
